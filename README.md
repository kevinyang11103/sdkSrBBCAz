# 高血压人群分析可视化系统 python1220

## 项目概述

本项目是基于 Python 的一个大数据分析系统，旨在为高血压人群的健康数据提供可视化展示。通过后端数据分析和前端可视化技术，为用户提供直观的数据洞察。

## 技术栈

### 后端

- **框架**: Django
- **Python 版本**: 3.7/3.8
- **数据库**: MySQL 5.7 或更高版本（兼容 5.X 系列）

### 数据处理

- **数据库工具**: Navicat 11（推荐使用）

### 开发环境

- **开发软件**: PyCharm 或 VS Code

### 前端

- **框架**: Vue.js

## 功能模块

- 用户管理
- 数据收集
- 数据分析
- 可视化展示
- 报告生成

## 系统角色

- 系统管理员
- 数据录入员
- 数据分析师
- 普通用户

## 运行环境

- 操作系统：Linux 或 macOS（推荐），Windows 理论上也支持
- Python 环境搭建：遵循 Python 官方指南

## 部署步骤

1. 安装 Python 环境
2. 安装依赖库和框架
3. 配置 MySQL 数据库
4. 部署前端 Vue.js 项目
5. 部署后端 Django 项目
6. 进行系统集成测试

## 目录结构

```
project/
├── backend/                # 后端 Django 代码
├── frontend/               # 前端 Vue.js 代码
└── scripts/                # 数据处理脚本
```

- 高效的数据处理与分析

- 灵活的报告生成与导出
